# View CopyTree Documentation

Display CopyTree's built-in documentation:

```bash
copytree copy:docs --display
```

This shows all available documentation in the terminal. You can also:
- List available topics: `copytree copy:docs --list`
- View specific sections: `copytree copy:docs --section <id>`
- View specific groups: `copytree copy:docs --group <id>`
- View task bundles: `copytree copy:docs --task <id>`

The documentation covers CLI usage, profiles, transformers, configuration, and troubleshooting.
